# Aguas - Water Management Application

A modern web application built with Next.js for water/aquatic resource management.

## 🚀 Technology Stack

- **Framework:** Next.js 15+ with TypeScript
- **Styling:** Tailwind CSS
- **Frontend:** React with App Router
- **Backend:** Next.js API Routes
- **Linting:** ESLint
- **Package Manager:** npm

## 📋 Prerequisites

- **Node.js:** v24.11.1 or higher (portable version)
- **npm:** 11.6.2 or higher

## 🛠️ Setup: Node.js Portable

Before running npm commands, configure the PATH to use portable Node.js:

```powershell
$env:PATH = "$env:USERPROFILE\Downloads\node-v24.11.1-win-x64;" + $env:PATH
```

## 🚀 Getting Started

### Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Production Build

```bash
npm run build
npm start
```

## 📁 Project Structure

- `/app` - Next.js App Router pages and layouts
- `/public` - Static assets
- `/components` - Reusable React components
- `/.github` - GitHub configuration

## 📝 Available Scripts

- `npm run dev` - Development server with hot reload
- `npm run build` - Production build
- `npm start` - Start production server
- `npm run lint` - Run ESLint checks
